# Institute of Digital Risk (IDR) – UI + Web Development Assignment
This project is a responsive single-page website created for the **Institute of Digital Risk (IDR)** as part of a UI + Web Development assignment.

The goal of the project is to design a simple brand identity and homepage that represents a modern digital risk training institute.


## Project Features
- Responsive single-page website
- Clean and minimal UI design
- Semantic HTML5 structure
- Sticky navigation bar
- Smooth scrolling navigation
- Service cards layout
- Contact form section
- Mobile-friendly layout


## Technologies Used
- HTML5
- CSS3 (Flexbox)
- JavaScript
- Google Fonts
